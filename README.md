
# React page admin

This is the admin page where you can add, edit, and delete any product. Additionally, it features customer support through chat


## Run Locally

Clone the project

```bash
  git clone https://github.com/chutrinh/e-commerce-admin.git
```

Go to the project directory

```bash
  cd e-commerce-admin
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Features

- add a new product
- delete a product
- edit a product
- chat with customers


## Tech Stack

**admin:** React, bootstrap

**Server:** Node, Express
